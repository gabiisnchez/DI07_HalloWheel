===================================================
   BIENVENIDO AL INSTALADOR DE HALLOWHEEL v1.0
===================================================

Gracias por elegir HalloWheel, la ruleta definitiva para tus fiestas de Truco o Trato.

REQUISITOS DEL SISTEMA:
- Sistema Operativo: Windows 10 o Windows 11 (64 bits).
- Espacio en disco: 100 MB libres.
- Memoria RAM: 2 GB mínimo.

NOTA IMPORTANTE SOBRE JAVA:
Esta aplicación es totalmente portable y autónoma. 
NO ES NECESARIO tener Java instalado previamente en su equipo. 
El instalador incluye una versión optimizada y ligera de Java Runtime Environment (JRE) configurada específicamente para este programa.

Antes de continuar, le recomendamos cerrar otras aplicaciones activas para garantizar una instalación rápida y sin errores.

¡Prepárese para girar la rueda!
